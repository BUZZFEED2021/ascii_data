===================
ASCII LIST
===================
install package:

.. code-block:: bash

    $ pip install connexion
    $ pip install connexion[swagger-ui]

Running:

.. code-block:: bash

    $ ./ascii_data.py

Now open your browser and go to http://localhost:9090/v1.0/ui/ to see the Swagger UI.

Test:

.. code-block:: bash

    $ ./test_ascii_data.py
